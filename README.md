# Tic-Tac-Toe-Game
A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.
This project is perfect for beginners to understand DOM manipulation, event handling, and basic game logic.
## Features
 2-player mode (Player X vs Player O)
 Turn-based gameplay
 Winner detection system
 Draw detection
 Restart game functionality
 Prevents overwriting filled cells
 Clean and modern UI
## Tech Stack
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and interactivity
